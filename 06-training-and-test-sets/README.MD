## 训练集和测试集 (Training and Test Sets)

预计用时：2 分钟

**测试集**是用于评估根据训练集开发的模型的数据集。

### 学习目标

- 了解将数据集分成训练集和测试集的优势。

### 拆分数据集

随机后进行拆分. 首先

- 训练集规模越大，模型的学习效果就越好。
- 测试集规模越大，我们对于评估指标的信心越充足，置信区间就越窄。

分成两个数据集：

- 训练集
- 测试集
    - 数据集比较大时, 测试集可以占(10%-15%)
    - 数据集比较小时, 我们可能需要执行诸如交叉验证之类较为复杂的操作

典型陷阱：**请勿对测试数据进行训练**

- 损失低得令人惊讶？
- 在庆祝之前，请检查您是否不小心对测试数据进行了训练
